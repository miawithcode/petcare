# Petcare

## Tech Stacks

- **Framework**: [Next.js](nextjs.org)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **UI**: [shadcn/ui](https://ui.shadcn.com/)
- **ORM**: [Prisma](https://prisma.io)
- **Database**: [SQLite](https://sqlite.org)
- **Payment**: [Stripe](ripe.com)
